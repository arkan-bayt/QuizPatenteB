============================================
  Quiz Patente B - دليل بناء APK للاندرويد
============================================

هذا الدليل يشرح كيفية بناء تطبيق APK
وتثبيته على جهازك الاندرويد فقط.
============================================

== الخطوة 1: تحميل البرامج ==

1) Node.js (مجاني)
   https://nodejs.org
   اختر LTS version → حمل → ثبّت

2) Android Studio (مجاني)
   https://developer.android.com/studio
   حمل → ثبّت
   أثناء التثبيت: اختر "Standard Installation"

== الخطوة 2: فتح المشروع ==

1) انسخ مجلد QuizPatenteB الكامل إلى حاسوبك
2) افتح Terminal (موجه الأوامر)
3) اذهب إلى مجلد المشروع:
   cd QuizPatenteB

4) ثبّت التبعيات:
   npm install

== الخطوة 3: تعديل رابط الموقع (مهم جداً!) ==

1) افتح الملف: capacitor.config.ts
2) غير الرابط إلى رابط موقعك الحقيقي على Vercel
   مثال:
   server: {
     url: 'https://اسم-مشروعك.vercel.app',
     cleartext: true
   }

== الخطوة 4: فتح المشروع بـ Android Studio ==

1) من Terminal داخل مجلد QuizPatenteB اكتب:
   npx cap open android

2) سيفتح Android Studio تلقائياً

== الخطوة 5: بناء APK ==

في Android Studio:

1) انتظر حتى ينتهي Gradle من التحميل (الشريط أسفل)
2) من القائمة العلوية:
   Build → Build Bundle(s) / APK(s) → Build APK(s)

3) انتظر حتى ينتهي البناء

4) ستظهر رسالة: "APK(s) generated successfully"
   اضغط "locate" لفتح مجلد APK

== الخطوة 6: نقل APK للهاتف ==

الملف موجود في:
android/app/build/outputs/apk/debug/app-debug.apk

طريقة النقل:
- عبر كابل USB: انسخ الملف إلى Downloads
- أو عبر Google Drive أو WhatsApp

== الخطوة 7: تثبيت على الهاتف ==

1) افتح "الإعدادات" على هاتفك
2) اذهب إلى: الأمان → تثبيت من مصادر غير معروفة
   (أو: التطبيقات → القائمة الثلاثة → الوصول الخاص)
3) فعّل الخيار
4) افتح ملف APK من ملفات الهاتف
5) اضغط "تثبيت"
6) افتح التطبيق! ✅

== ملاحظات مهمة ==

- التطبيق يحتاج إنترنت لأنه يرتبط بـ Supabase
- الرابط في capacitor.config.ts يجب أن يكون صحيحاً
- كل مرة تغير شيئاً في الموقع، تحتاج إعادة بناء APK
- الحجم المتوقع للـ APK: حوالي 15-20 MB

== مشاكل وحلول ==

مشكلة: Gradle يفشل
حل: افتح Android Studio → SDK Manager → تأكد
     أن Android SDK 34 مثبت

مشكلة: التطبيق يفتح صفحة بيضاء
حل: تأكد أن الرابط في capacitor.config.ts صحيح
     وأن الموقع يعمل على المتصفح

مشكلة: "app not installed"
حل: احذف النسخة القديمة أولاً ثم ثبّت الجديدة
